# Poem-generator
A machine learning project about poem generation and LLM.
A machine learning project about poem generation and LLM. I've done this project entirely on Python, it is a machine learning algorithm that, given an input, returns a poem based on the context of that input. Moreover, I've used three different ML methods to see which one performs best, those methods are: LSTM(Long Short-Term Memory), RNN(Recurrent Neural Network) and GRU(Gated Recurrent Unit). The models are trained on a kaggle dataset containing poems.
